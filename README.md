## Installation

Add this line to your application's Gemfile:

```ruby
gem "skeletor"
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install skeletor
```

## Usage

puts Skeletor::VERSION # => "1.0.0"
